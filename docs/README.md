# Документация репозитория Control Plane

Материалы для разработчиков и для текстов в интерфейсе (разделы «О платформе», «Обучение» могут ссылаться на эти файлы в вашей системе контроля версий или Wiki).

| Файл | Назначение |
|------|------------|
| [`first_vkr_draft.md`](first_vkr_draft.md) | Черновик ВКР: контекст ЦКП СКИФ, цифровые двойники, обоснование платформы, архитектура backend/frontend, функциональные подсистемы, модель данных (ссылка на DBML). |
| [`cplane.dbml`](cplane.dbml) | Схема PostgreSQL в формате DBML (миграции `000001`–`000006`). Импорт в [dbdiagram.io](https://dbdiagram.io): **Database → Import → DBML**. |

## Связь с интерфейсом

Стартовые тексты для экранов «О платформе», «Обучение» и демо-записи «Обновления» задаются при первом запуске из каталога [`backend/internal/bootstrap/cms/defaults/`](../backend/internal/bootstrap/cms/defaults/). После деплоя их можно заменить в UI (режим администратора) или правкой файлов и пересборкой backend.

## Рекомендуемые внешние ссылки для блока «О платформе»

Подставьте вместо заглушек URL вашей организации:

- страница Wiki или Confluence с пользовательским руководством;
- GitLab/GitHub с каталогом `docs/` (этот README или отдельные гайды);
- канал поддержки или трекер задач.
